# Project Name: DigiTools


##  Technology Stack:
    Frontend: React.js
    Styling: Tailwind CSS and Daisy UI
    State Management: React Hooks (useState, use)
    Notifications: React-Toastify
    Icons: React-Icons

## Key Features
    ### 1. Dynamic Toggling System: 
    Toggle Product and Cart using Conditional-Rendering without page reload.
    ### 2. Cart Management: 
    Users can add products, remove products,  and remove all products with one click. Also, users can see live count in navbar.
    ###  3. Interactive design: 
    Design all sections interactively and   divide all  rows into 3 columns   so that it looks good.
